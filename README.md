PawMart - Pet Adoption & Supplies Platform

PawMart is a web-based platform that connects pet lovers with pets available for adoption and pet owners or stores selling pet-related supplies. This platform enables users to browse listings, place orders, and manage their own pets or productsâ€”all in a secure and user-friendly interface.

1. About the Website

PawMart is designed to make pet adoption and the purchase of pet supplies easier and more accessible. Users can:

Browse pets available for adoption and pet products for sale.

View detailed listing information including category, price, location, and owner contact.

Place orders or adoption requests through a simple form.

Manage their own listings and orders in one place.

2. Purpose of the Website

The primary purpose of PawMart is to:

Help pet lovers find pets for adoption safely.

Provide a platform for sellers or pet owners to showcase pets and pet-related products.

Streamline the adoption and purchasing process through an easy-to-use interface.

3. Importance & Need

Accessibility: Connects people who want to adopt pets with those offering them.

Efficiency: Reduces the hassle of searching multiple places for pets or products.

Transparency: Shows detailed listing information and owner contact.

Organization: Users can track their listings and orders in one place.

Digital Record: Order history and reports can be downloaded as PDFs for reference.

4. Technology Stack

The website was built using modern web technologies for a responsive and interactive experience:

Frontend: React.js

Routing: React Router DOM

Styling: Tailwind CSS

State Management: React Context API

Data Storage: LocalStorage (for development/demo)

Utilities:

react-hot-toast (for notifications)

jsPDF + jsPDF-AutoTable (for PDF reports)

Optional future backend integration can use Firebase or MongoDB for persistent storage.

5. Features

User Authentication & Private Routes: Secure access to personal listings and orders.

Pet & Product Listings: Browse listings with detailed information.

Order Management: Place adoption requests or purchase orders with auto-filled information.

Listing Management: Add, update, or delete your own listings.

Export Reports: Download order history as PDF files.

Responsive Design: Works seamlessly on mobile, tablet, and desktop devices.

Error Handling & Notifications: 404 page, toast notifications, and loading spinners for better UX.
